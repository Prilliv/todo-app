# Todo App

Простий застосунок для відображення списку задач, створений за допомогою React та Vite.

## Функціонал
- відображення списку задач
- список задач заданий як константа
- використано React Props
- список рендериться через map
- застосунок складається з декількох компонентів
- додано базове стилізування

## Структура проєкту
- App.jsx — головний компонент
- TodoList.jsx — компонент списку задач
- TodoItem.jsx — компонент окремої задачі

## Запуск проєкту

```bash
npm install
npm run dev
